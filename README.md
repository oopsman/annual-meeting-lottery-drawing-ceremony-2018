# 九次方大数据年会抽奖程序（2018）

应公司要求，九次方大数据可视化部为2018年公司年会的抽奖环节开发了一款高逼格的抽奖程序。   
针对抽奖业务，随机算法优先使用random.org相关真随机数接口，网络不可达时调用JavaScript本身的伪随机数生成方法。   
针对视觉美学，与九次方大数据可视化部此次年会宣传PPT风格保持统一，采用抽象线条树和WebGL Shader实现数据星河光影效果。
